## car-order-admin

### 项目启动

- npm install
- npm install -g commitizen@4.2.4
- npm run dev

### 项目简介

预约用车后台管理系统

### 技术栈

- 框架: [Vue3](https://cn.vuejs.org/)
- 打包工具: [Vite](https://cn.vitejs.dev/)
- 组件库: [Element Plus](https://element-plus.org/zh-CN/)

### 开发环境

- Windows 版本: Windows 10 专业版
- 开发工具: VSCode
- Node 版本: v18.12.1
- npm 版本: v8.17.0
